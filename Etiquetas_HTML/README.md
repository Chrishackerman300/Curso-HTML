# Etiquetas

## ¿Qué son las etiquetas en html?

Las etiquetas HTML son pequeños bloques de código que le indican al navegador como debe interpretar el contenido de nuestra web.

Las etiquetas HTML comienzan siempre con el símbolo **(<)** y finalizan con el símbolo **(>)** y entre estos símbolos entra el nombre de la etiqueta que el navegador interpretará.

Todas las etiquetas HTML están compuestas por una **Etiqueta de apertura y Etiqueta de cierre**. Aunque hay excepciones, como por ejemplo la etiqueta para cargar una imagén, estás etiquetas se hacen llamar **Etiquetas Huérfanas**, ya que solo tienen una etiqueta de apertura.

Las etiquetas de cierre se contruyen con barra **(/)** justo antes del nombre de la etiqueta. La sintaxis básica de una creación de etiqueta HTML es la siguiente:

### Sintaxis

```html
<Etiqueta></Etiqueta>
```
